1 - Présentation

Ce plugin vous permet de générer automatiquement un numéro d’inventaire.

Compatibilité 0.71.

2 - Installation

Récupérez l'archive tar.gz du plugin
Décompressez la dans le répertoire plugins de votre installation de glpi

Une fois copié dans le répertoire, dans l'onglet plugin vous verrez apparaître "Génération automatique du numéro d'inventaire".
La gestion des droits se fait directement depuis Administration -> Profils.
